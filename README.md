#  Superstore Sales Analysis using SQL

##  Project Overview
This project performs end-to-end sales data analysis using MySQL.
The objective is to extract meaningful business insights from raw Superstore sales data.

---

##  Objectives
- Analyze total sales and profit performance
- Identify top customers using RANK()
- Detect loss-making products
- Perform category-wise profit analysis
- Analyze monthly sales trends

---

## Tools Used
- MySQL
- SQL Window Functions
- Data Cleaning Techniques

---

##  Key Insights
- Technology category generated highest profit
- Some products consistently resulted in losses
- Top customers contributed major revenue share
- Clear seasonal trends observed
